使用 Hexo Admin 插件

使用 Hexo Admin 插件

Hexo Admin 是一个本地在线式文章管理器，可以用直观可视化的方式新建、编辑博客文章、page页面，添加标签、分类等，并且支持剪贴板粘贴图片（自动在source_images_目录中创建文件）

使用方法

  1. 在Hexo网站目录下，安装 Hexo Admin 插件
  2. 启动本地服务器并打开管理界面，即可使用
